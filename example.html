Github Badges

Load jquery in the page followed by the github badge script:

<script src="github-badges.js"></script>

Open Issues

Place an image with a class github-badge and a src attribute with the following scheme:

"github-badges/username/repository/issues"

For example the following img element:

<img src="github-badges/psas/psas_packet/issues" class="github-badge">
